### GDB在Mac上的安装过程以及使用前遇到的问题  
#### 安装过程  
  1.首先，使用`brew search gdb`搜索是否已经安装gdb，如果没有，则按照如下步骤进行安装  
  2.其次执行`brew install gdb`进行安装（默认是最新版本）  
  3.安装完毕后使用`gdb-v`查看是否安装成功
